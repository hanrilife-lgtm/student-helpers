Предметы

Правила пользования

1. Сервис предоставляет доступ к учебным материалам и ИИ-репетитору.

2. Запрещено использовать сервис для списывания на экзаменах.

3. Запрещено загружать вредоносный код или контент.

4. Администрация оставляет за собой право блокировать аккаунты за нарушение правил.

5. Все материалы защищены авторским правом.

Политика конфиденциальности

1. Мы собираем только email для работы сервиса.

2. Пароли не используются — вход по одноразовому коду.

3. Мы не передаём данные третьим лицам.

4. ИИ-репетитор использует Google Gemini API — ваши запросы отправляются на серверы Google.

5. Вы можете удалить аккаунт в любой момент, все данные будут удалены.

6. Мы используем куки только для хранения темы и API-ключа (локально).
